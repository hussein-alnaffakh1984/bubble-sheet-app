# 🎯 نظام تصحيح أوراق الإجابة الآلي (Bubble Sheet Remark)

نظام متكامل مبني بـ **Python + Flask + OpenCV** لتصحيح أوراق الإجابات (Bubble Sheets) آلياً.

## ✨ المميزات

- 📝 **إنشاء اختبارات مرنة**: حدد عدد الأسئلة (1-300)، الخيارات (2-5)، عدد أرقام رقم الطالب
- 📄 **توليد ورقات إجابة PDF** جاهزة للطباعة مع علامات محاذاة احترافية
- 🤖 **تصحيح آلي بـ OpenCV**: تصحيح المنظور، كشف الدوائر بـ HoughCircles، تحديد الإجابات المظللة
- 🆔 **قراءة رقم الطالب** تلقائياً من خانات التظليل
- 📊 **إحصائيات شاملة**: المعدل، النجاح/الرسوب، تحليل صعوبة الأسئلة
- 📑 **تصدير Excel** بأربع أوراق: ملخص، تفاصيل، تحليل أسئلة، نموذج إجابة
- ➖ **دعم الخصم على الإجابة الخاطئة** (Negative Marking)
- 🌐 **واجهة عربية احترافية** مع تصميم حديث
- 📦 **رفع متعدد**: عدة صور دفعة واحدة

## 🚀 التشغيل

```bash
cd bubble_sheet_app
python app.py
```

ثم افتح المتصفح على: http://localhost:5000

## 📋 خطوات الاستخدام

### 1️⃣ إنشاء اختبار جديد
1. اذهب إلى "اختبار جديد"
2. أدخل العنوان، عدد الأسئلة، الخيارات، إعدادات الدرجات
3. حدد نموذج الإجابة الصحيحة (يمكن الملء العشوائي للاختبار)
4. اضغط "إنشاء الاختبار" - سيتم توليد PDF تلقائياً

### 2️⃣ طباعة وتوزيع
- حمل ملف PDF
- اطبعه وأعطه للطلاب لحل الأسئلة
- اطلب منهم تظليل الدوائر بقلم رصاص داكن

### 3️⃣ مسح وتصحيح
- بعد جمع الأوراق، امسحها بسكانر (200 DPI أو أعلى مفضل)
- من قائمة "الاختبارات" → اضغط أيقونة الكاميرا
- ارفع الصور (يمكن عدة دفعات معاً)
- اضغط "بدء التصحيح" - النتائج فورية

### 4️⃣ مراجعة النتائج
- اعرض النتائج الفردية
- تصدير Excel للنتائج الكاملة
- تحليل الإحصائيات وصعوبة الأسئلة

## ⚙️ التقنية

- **OpenCV HoughCircles** لكشف الدوائر بدقة
- **Perspective Correction** لتصحيح الميل والمنظور
- **Adaptive Thresholding** لتحديد التظليل
- **Cluster Analysis** لتجميع الدوائر في صفوف وأعمدة

## 💡 نصائح للحصول على أفضل نتائج

1. **استخدم قلم رصاص داكن (HB أو 2B)** أو قلم جاف أسود
2. **املأ الدوائر بالكامل** ولا تضع علامات صغيرة
3. **امسح الأوراق بدقة عالية** (200 DPI على الأقل)
4. **تأكد من ظهور علامات الزوايا السوداء** في الصورة (مهمة لتصحيح المنظور)
5. **لا تضع علامات خارج الدوائر**
